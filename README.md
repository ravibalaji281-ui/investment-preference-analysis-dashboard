# investment-preference-analysis-dashboard📊 Investment Preference Analysis Dashboard

📌 Project Overview

This project analyzes investment preferences using Power BI. The dashboard provides interactive visualizations that help understand investor behavior, preferred investment options, risk tolerance, and demographic trends.

🎯 Objectives

- Analyze investment preferences across different groups.
- Identify popular investment options.
- Understand investor risk profiles.
- Create an interactive dashboard for better business insights.

🛠️ Tools Used

- Power BI
- Microsoft Excel
- Power Query
- DAX

📈 Dashboard Features

- Investment distribution analysis
- Age-wise investment preferences
- Gender-wise comparison
- Risk appetite analysis
- Interactive slicers and filters
- KPI cards and charts

📂 Project Files

- Power BI Dashboard (.pbix)
- Dataset (.xlsx/.csv)
- Dashboard Screenshots
- README.md

📊 Key Insights

- Identifies the most preferred investment options.
- Compares investment choices across different age groups.
- Highlights investor risk behavior.
- Supports data-driven financial decision-making.
